# 입점 파트너 신청 자동메일 설정

이 자동화는 파트너 신청 Google Form의 **응답 스프레드시트**에 연결합니다.

## 처음 한 번만 설정

1. 파트너 신청 Google Form에서 `응답` 탭을 엽니다.
2. 초록색 Sheets 아이콘을 눌러 응답 스프레드시트를 엽니다.
3. 스프레드시트에서 `확장 프로그램 → Apps Script`를 엽니다.
4. 기본 코드를 지우고 `partner-form-auto-email.gs` 내용을 붙여넣어 저장합니다.
5. 스프레드시트를 새로고침합니다.
6. 상단 메뉴에서 `Harmony Link 자동화 → 1. 접근코드 저장`을 선택합니다.
7. 홈페이지와 동일한 접근코드를 입력합니다.
8. `Harmony Link 자동화 → 2. 자동메일 시작`을 선택하고 권한을 승인합니다.

## 작동 방식

- 신청자가 Google Form을 제출하면 신청자 이메일로 자료실 주소와 접근코드가 발송됩니다.
- `hibelle@hibelleconsulting.com`에도 새 신청 내용과 자동발송 완료 알림이 전송됩니다.
- 이메일 질문 제목에 `이메일`, `Email`, `메일` 중 하나가 포함되어 있어야 합니다.

## 중요한 운영 안내

이 설정은 신청 즉시 자료실 접근코드를 보냅니다. 심사 승인 후에만 보내려면 자동 트리거 대신 관리자 승인 열을 사용하는 별도 승인형 흐름으로 변경해야 합니다.

정적 홈페이지의 접근코드는 강한 보안 로그인 수단이 아닙니다. 민감한 개인정보나 계약 원본은 자료실에 올리지 마세요.
